# シンデレラガールズのライブ出演者からオリジナルメンバーの揃い具合を調べるツール

## Prerequisites

```sh
npm install
```

## Dev

```sh
npm run dev
```

## Build

```sh
npm run build
```

## Deploy

```
git push origin master
```
